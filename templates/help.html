Getting Started

Basic Usage

  1. Enter your Arabic chat text in the input field
  2. Select the appropriate dialect (currently Moroccan Arabic)
  3. Click "Convert to Arabica" to see the academic transliteration
  4. Use the "Copy" button to copy the transliteration to your clipboard

Example

Arabic Chat Input
mar7aba, kayf 7alek?
Arabica Transliteration
marḥaba, kayf ḥalek?

The Arabica System

The Arabica transliteration system is used when a very strict transliteration of Arabic into Latin characters is required. In this system, each Arabic consonant corresponds to a specific Latin letter, sometimes with diacritical marks.

Key Principles

  • Orthography is prioritized over pronunciation
  • Solar letters are transcribed without assimilation
  • Diacritical marks are used to distinguish between similar consonants

Arabic Chat Alphabet

The Arabic chat alphabet (also known as Arabizi, Franco-Arabic, or Arabish) is an informal way of writing Arabic using Latin characters and numbers. It's commonly used in digital communication like text messages, social media, and online chat.

Moroccan Arabic Features

Moroccan Arabic (Darija) has several distinctive features that may affect transliteration:

Pronunciation Differences

  • The letter ق (qāf) is often pronounced as "g" in many Moroccan dialects
  • Short vowels are often reduced or omitted
  • Influenced by Berber, French, and Spanish, with many loanwords

Chat Conventions

Common conventions in Moroccan Arabic chat include:

  • Using "g" for both ج and ق depending on pronunciation
  • Using "ch" for ش (influenced by French)
  • Including French and Spanish loanwords with "p" and "v" sounds

Examples

Moroccan Chat Arabica English
ch7al šḥal How much
goul gūl Say (where ق is pronounced as g)
wa5a waḫa Okay

Conversion Rules

ArabiChat applies several rules when converting Arabic chat to Arabica transliteration:

Character Mapping

  • Numbers and letter combinations are mapped to their Arabica equivalents
  • Repeated vowels (e.g., "aa") are converted to long vowels (ā)
  • Capital letters may be used to indicate emphatic consonants (e.g., "S" for ص)

Special Rules

  • The definite article is always written as "al-" and connected with a hyphen
  • The tāʾ marbūṭa is rendered as "a" at the end of words, but "at" in construct state
  • Sun letters are not assimilated in the transliteration

Tāʾ Marbūṭa madina_t al maghrib madīnat al-maġrib
Long Vowels kitaab kitāb

Troubleshooting

Common Issues

Problem: Some characters are not being converted correctly.

Solution: Check if you're using non-standard chat conventions. Try using a more standardized representation:

  • Use "3" for ع instead of a different number
  • Use "7" for ح instead of a different number
  • Capitalize emphatic consonants (T for ط, S for ص, etc.)

Problem: Short vowels are missing in the output.

Solution: Add explicit vowels in your chat text:

  • Use "a", "i", and "u" for short vowels
  • Use "aa", "ii", and "uu" for long vowels
  • Be explicit about vowels that might be ambiguous in chat language

Problem: Certain dialectal features are not transliterated correctly.

Solution: Be explicit about dialectal features:

  • Use "g" for the Moroccan pronunciation of ق
  • Use "q" when you want to represent the standard Arabic ق
  • Consider adding notes or annotations for particularly ambiguous cases

Getting Help

If you encounter issues not covered here, you can:

  • Check the GitHub repository for updates and known issues
  • Submit an issue on GitHub with a detailed description of the problem
  • Include examples of the text that's causing problems
